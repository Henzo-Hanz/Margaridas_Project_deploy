# Margaridas_garden
O objetivo é criar um gerenciador de senhas amigável com o usuário, com uma alta segurança e confiabilidade,utilizando Python e FastApi. O projeto contará com auxílio de IA para a criação do projeto com Agentes como Gpt-codex,Sonnet 4.5, Compose 1.0 e etc... Presentes na IDE Cursor, servindo como Projeto "educacional"  e Prática no Uso de Agentes.
